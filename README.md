# Varshu's Art

> A React based web application for displaying the art gallery of my sister - Jaya Varshini

## Tech Stack
- HTML
- SASS
- Bootstrap
- React JS

## Dependencies
1. [Node Js](https://nodejs.org/en/download/)

## Getting Started
1. Install dependencies  
```
npm install
```
2. Launch the development server  
```
npm start
```
